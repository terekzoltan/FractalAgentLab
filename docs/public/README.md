# Public Docs Staging

This directory is the private repo staging surface for public-sanitizable documentation.

Rules:
- material here is still drafted and versioned in the private canonical repo first
- nothing here syncs automatically to a public repo
- every export still requires Meta `visibility-release-review`
- do not copy `ops/` or `docs/private/` content here without deliberate sanitization

Use this directory for:
- public-safe architecture summaries
- workflow explainers
- polished design notes intended for later public mirroring
